## Estructura fundamental del documento html 
- ´<!DOCTYPE html>´ - Declaramos el tipo de 
- documento, este elemento huerfano indica el navegador que el documento con con el que se esta trabajando y que debera renderizar es ´html´, **Siempre debe estar en la primera linea**.

## Estructura de contenido semantico (seciones principales)

Estructura de Contenido Semántico
1. Encabezado (header)
Es la parte de arriba de la página web. Aquí normalmente se coloca el título, el logo y el menú principal.
2. Navegación (nav)
Se usa para poner los enlaces o el menú que ayuda a moverse dentro de la página.
3. Contenido Principal (main)
Es la parte más importante de la página, donde va la información principal.
4. Sección (section)
Sirve para dividir el contenido en diferentes partes u temas relacionados.
5. Artículo (article)
Se utiliza para contenido independiente como noticias, publicaciones o artículos.
6. Barra Lateral (aside)
Contiene información extra, anuncios o enlaces relacionados.
7. Pie de Página (footer)
Es la parte final de la página web. Generalmente contiene información de contacto, derechos de autor o redes sociales.
Importancia del Contenido Semántico
Ayuda a organizar mejor la página web.
Hace que el código sea más fácil de entender.
Mejora la accesibilidad para los usuarios.
Ayuda a que la página aparezca mejor en los buscadores como Google.
